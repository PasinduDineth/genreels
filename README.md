# Genreels

MVP TikTok shorts generator built with React, Node.js, Pollinations, and Remotion.

## Planned flow

1. Enter a story topic.
2. Generate 10 constrained image prompts with Pollinations text generation.
3. Generate 10 images from those prompts with Pollinations image generation.
4. Render a silent 9:16 short with Remotion.
5. Preview prompts, images, and the final video in the UI.
